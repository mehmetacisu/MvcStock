The aim of the project is to teach the mvc 5 structure and architecture, to gain the ability to develop projects with mvc.

<p>Frontend : Bootstrap</p> 
<p>Backend  : .NET MVC5</p>
<p>Database : MSSQL</p>
<p>ORM : Entity Framework 6</p>
